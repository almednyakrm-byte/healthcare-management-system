# نظام إدارة صحي مع سجلات مريض وتحليلات طبية وإحصائيات
==========================

## ملخص المشروع
----------------

نظام إدارة صحي مع سجلات مريض وتحليلات طبية وإحصائيات هو تطبيق إلكتروني يهدف إلى تسهيل إدارة السجلات الطبية وتحليلاتها وتقديم إحصائيات متكاملة لتحسين جودة الرعاية الصحية.

## هيكل المشروع
----------------

### مجلد المشروع

* `app`: مجلد تطبيق النظام
* `docker`: مجلد Docker
* `docs`: مجلد الوثائق
* `tests`: مجلد الاختبارات

### مجلد تطبيق النظام (app)

* `models`: مجلد النماذج (tables)
* `controllers`: مجلد التحكم
* `services`: مجلد الخدمات
* `routes`: مجلد الطرق

### مجلد Docker (docker)

* `docker-compose.yml`: ملف Docker Compose
* `Dockerfile`: ملف Docker

## إعداد بيئة التطوير باستخدام Docker Compose
---------------------------------------------

### خطوات الإعداد

1. تأكد من توفر Docker و Docker Compose على جهازك.
2. افتح مجلد المشروع في محطة العمل.
3. افتح terminal وانتقل إلى مجلد المشروع.
4. استخدم الأمر التالي لتشغيل بيئة التطوير:
bash
docker-compose up

5. انتظر حتى يُنشئ Docker بيئة التطوير.
6. افتح متصفح ويب ومرر إلى `http://localhost:3000` لفتح تطبيق النظام.

## وحدات النظام
----------------

### وحدات البيانات (tables)

* `patients`: سجلات المرضى
* `doctors`: سجلات الأطباء
* `appointments`: مواعيد الزيارات
* `medications`: أدوية المرضى

### وحدات التحكم (controllers)

* `PatientController`: التحكم في سجلات المرضى
* `DoctorController`: التحكم في سجلات الأطباء
* `AppointmentController`: التحكم في مواعيد الزيارات
* `MedicationController`: التحكم في أدوية المرضى

### أدوار النظام (roles)

* `admin`: مدير النظام
* `doctor`: طبيب
* `patient`: مريض
